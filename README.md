# knowhub

## Quick Start

1. `npm install`
2. `npm run build` (core)
3. `npm run dev` (core)
4. `npm run dev:web` (web)
5. `npm run lint`

### Windows/OneDrive Note

If `apps/web` fails with `spawn EPERM`, use `npm run dev` and `npm run dev:web` in separate terminals and treat `npm run build:web` as an environment-conditional step.
